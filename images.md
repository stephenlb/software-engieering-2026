## Usage Notes

All prompts use a consistent **flat infographic colorful business-oriented design** style for visual cohesion across the document.

### Platform-Specific Tips

- **Gemini**: Works well with these detailed prompts. The flat infographic style translates well.
- **OpenAI DALL-E**: May need to simplify longer prompts. Keep the "flat infographic colorful business-oriented design" suffix.
- **Midjourney**: Add style suffixes like `--style raw` or `--v 6` for cleaner technical diagrams.

### Optional Modifiers

For all prompts, consider adding:
- "White background" for documentation use
- "16:9 aspect ratio" for presentations
- "No text labels" if you want to add text yourself in post-processing
- "Vector style" for scalable graphics
